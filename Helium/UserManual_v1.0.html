Basics

HtmlLib is Java library of HTML elements.
HtmlLib is intended to build web sites, so all methods are used to set some values, rather to modify or delete them.

Tools

HtmlLib provides various tools that simplify HTML document creation.

DynamicTableBuilder

DynamicTableBuilder tool is used for creating tables with customized contents. Table fields can contain any HTML elements or any other object that implements toString method.
Columns and rows can be added to the table. Rules for building tables.

SqlTableBuilder

SqlTableBuilder is concrete implmentation of DynamicTableBuilder tool. SqlTableBuilder creates table on the basis of ResultSet object.

Helium

Purpose of Helium is to become a framework for building graphical user interfeces of web applications in Java. It supports generation of html documents (web pages), CSS styling, JavaScript support (including jQuery) and many tools (eg. for building tables from SQL queries).

Version history

0.0.1

december 2013

  • map of attributes moved into new class Attributes
  • map of styles moved into new class Styles

0.0.0

december 2013

  • project HtmlLib renamed to Helium

1.1.0

september 2013

  • introduction of CSS styling
  • styles can be applied as inline styles
  • styles can be applied for element's id attribute
  • styles can be applied for element's class attribute
  • templates for Text styling, Box styling, Table styling, Background styling, Image styling and Link styling
  • Css class containing CSS attributes
  • Color class containing CSS color names
  • FontFamily class containing fonts
  • Text class containing Font-Decorations, Alignment...
  • Attributes of html elements moved to Attribute class
  • New elements: header, section, article, figure, figcaption, footer, aside (HTML5 only), nav, ul, li

1.0.2

august 2013

  • added Th, Script and Span elements
  • added more examples
  • renamed Break class to Br
  • renamed HorizontalRule class to Hr
  • DynamicTableBuilder implementation
  • SqlTableBuilder implementation
  • optimised toString method

1.0.1

july 2013

  • added more javadoc
  • added some examples
  • added Bold, Italic, Strong and Title element
  • added Anchor constructor with String attribute
  • removed empty constructor of Heading class
  • added DOCTYPE definitions in HtmlDocument class
  • corrected incorrect name of input element
  • attributes were missing in Body, Br, Heading and Paragraph elements
  • added missing build.xml file into tar.gz archive

1.0.0

june 2013

  • birth of HtmlLib project
  • supported elements: Anchor, Body, Br, Div, Form, Head, Heading, Hr, Img, Input, Link, P, Style, Table, Td, Tr
  • abstract element HtmlElement, which contains major methods for adding child elements and attributes
  • supported major attributes
  • some javadoc comments
  • build.xml file for generating jar file, javadoc and tar.gz archive
